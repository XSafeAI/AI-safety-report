# T2I-Safe

T2I-Safe is a modular evaluation toolkit for assessing the safety of T2I models.
It provides a unified framework to measure model behavior under benchmark, adversarial, and regulatory-compliance settings, supporting systematic analysis of visual risk, robustness, and alignment.

## Repository Structure
```
t2i-safe/
├── benchmark/   # Standard safety benchmark evaluation
├── adversarial/    # Adversarial evaluation
├── compliance/   # Regulatory compliance evluation
└── README.md     # Project overview (this file)
```
